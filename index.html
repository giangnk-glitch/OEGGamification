Kết nối Apps Script
Dashboard kết nối qua Google Apps Script Web App — không bị CORS, không cần publish CSV. Làm theo 3 bước bên dưới.
1
Mở Google Sheets → Apps Script
Trong file Gamification_Unified: Extensions → Apps Script → paste nội dung file Code.gs vào → Save
2
Deploy làm Web App
Deploy → New deployment → Web App
Execute as: Me · Who has access: Anyone → Deploy → Copy URL
3
Dán Web App URL vào đây
URL dạng: https://script.google.com/macros/s/XXXXXX/exec

Hướng dẫn Apps Script
Bước 1: Mở Google Sheets → Extensions → Apps Script
Bước 2: Paste nội dung file Code.gs vào editor → Save (Ctrl+S)
Bước 3: Deploy → New deployment → Web App
   Execute as: Me  ·  Who has access: Anyone
Bước 4: Copy URL → Dán vào màn hình setup → Kết nối
Lưu ý: Mỗi lần sửa Code.gs cần Deploy → New deployment (không phải Manage existing)
