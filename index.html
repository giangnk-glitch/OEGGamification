Kết nối Google Sheets
Dashboard sẽ đọc dữ liệu trực tiếp từ Google Sheets của bạn qua CSV public URL. Làm theo 3 bước dưới đây để kết nối, hoặc nhấn Dùng dữ liệu demo để xem trước.
1Publish file Google Sheets thành CSV
Mở file → File → Share → Publish to web → chọn từng sheet → format CSV → Publish. Lấy link dạng:
https://docs.google.com/spreadsheets/d/SHEET_ID/gviz/tq?tqx=out:csv&sheet=TÊN_SHEET
2Dán Sheet ID vào ô bên dưới
Sheet ID là chuỗi ký tự dài trong URL của file. Ví dụ: 1lYyc44bzIWpnt7oCXzTiqJk5scB9lsAE

Hướng dẫn Publish CSV từ Google Sheets
Bước 1: Mở Google Sheets → File → Share → Publish to web
Bước 2: Chọn từng sheet → chọn định dạng Comma-separated values (.csv)
Bước 3: Nhấn Publish → Copy link → Dashboard dùng link đó tự động
Lưu ý: Dashboard tự fetch mỗi khi nhấn "Làm mới". Để tự động cập nhật, bật Automatically republish when changes are made trong Google Sheets.
